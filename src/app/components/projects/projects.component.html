PokeApi Project Proyecto Api Pokemon

PokeAPIProject es un proyecto demostrativo. En este proyecto, se utiliza la PokeAPI para permitir a los usuarios explorar información detallada sobre diferentes Pokémon. Este proyecto se desarrollo en Angular usando PrimeNG y se encuentra alojado en Firebase Hosting.


Fonoteca de las infancias rurales de Colombia Fonoteca de las infancias rurales de Colombia.

La fonoteca es un proyecto desarrollado con la Universidad Pedagógica Nacional, en el marco de la investigación – creación DSI 592-22. La Fonoteca de las infancias rurales contiene el patrimonio sonoro, la memoria histórica, colectiva y cultural, de los saberes que circulan en las prácticas de crianza de niñas y niños en contextos rurales. Este proyecto se desarrollo en WordPress.
